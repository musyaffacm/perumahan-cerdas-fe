# INSTALASI

 1. Clone Repository
    ```bash
        git clone https://github.com/musyaffacm/perumahan-cerdas-fe.git
    ```

 2. Masuk ke Direktori Proyek
    ```bash
        cd perumahan-cerdas-fe
    ```

 3. Instal Dependensi
    ```bash
        yarn
    ```

 4. Salin Berkas .env
    ```bash
        yarn dev
    ```
